================================
Welcome to EPAC's documentation!
================================

Embarrassingly Parallel Array Computing: EPAC is an array-based workflow builder for machine learning tasks.

- Easily building machine learning workflow.
- Run EPAC tree in parallel on local multi-core machine or on HPC.
- Design your own machine learning algorithm as a plug-in node in EPAC tree.

These three features will be shown in introduction_. The installation is shown in installation_. To learn how to use EPAC, please start with tutorials_ which contains basic examples.
