To run all the tests and doctests of pylearn-epac at once:

-> Download nosetests at : http://pypi.python.org/pypi/nose/1.3.0

-> Unzip, untar and install nosetests with : python setup.py install --user

-> In the pylearn-epac/epac directory, run :
    nosetests -s --with-doctest --nologcapture --ignore-files=benchmark_mem.py
